Description

Automatically evaluating dialog response is a challenge task. Because you need to be able to determine if the response to the context is appropriate, but there are so many possible responses. Currently used methods include BLEU, ROUGE, and BERTSCORE. Word-overlap methods such as BLEU and ROUGE indicate how overlapping the correct answer is, BERTSCORE indicates how semantically similar it is to the correct answer. We created a demo to show how these metrics represent scores. In this demonstration, you can select three responses to the context to check the score. And you can also check the score for the input that the user responded to.

However, there is a limit to this metric that it is different from the actual score assigned by the actual person. Therefore, to overcome these limitations, we are studying metrics that take into account several aspects such as topical, emotional, and factual consistency. It has not yet been added to this demonstration because research is ongoing. If you want to hear more details, please contact us through the contact below.
